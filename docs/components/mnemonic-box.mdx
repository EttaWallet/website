A handy container for your Bitcoin seed phrase. Great for asking user to enter their seed phrase or to back it up manually

## How to use it

```jsx
import { MnemonicBox } from 'etta-ui';

// Input container
<MnemonicBox 
    value="yam flower delight seed cat note gritty pirate vault bean time loud"
    mode="INPUT"
/>;

// Readonly container
<MnemonicBox 
    value="yam flower delight seed cat note gritty pirate vault bean time loud"
    mode="READONLY"
/>;
```

## Props

<div className='table-responsive'>

| Name           | Type                                   | Default   | Description                                                        |
| -------------- | -------------------------------------- | --------- | ------------------------------------------------------------------ |
| `value`     | `string`        |           | The seed phrase as a string separated by spaces. E.g from BDK           |
| `mode` | `INPUT` \| `READONLY`            | Render mnemonic box as editable input container or Readonly seed container |
| `onChangeText?`     | `function`: () => void          | Define what happens once seed is updated (INPUT mode) |

</div>
