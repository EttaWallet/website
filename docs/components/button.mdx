A touchable, interactive element that initiates an action and or event. Buttons display text and can also include icons.

## Usage

```jsx
import { Button } from 'etta-ui';

<Button
  title="Click me"
  size="default"
  appearance="filled"
  icon="icon-bitcoin-circle"
  iconPosition="left"
/>;
```

## Variants

### Appearance

Buttons can appear filled, outline or transparent

<View style={{ flexDirection: 'row', alignItems: 'center'}}>
  <Button title="Filled" appearance="filled" style={{ marginHorizontal: 10 }} />
  <Button title="Outline" appearance="outline" style={{ marginHorizontal: 10 }} />
  <Button title="Transparent" appearance="transparent" style={{ marginHorizontal: 10 }} />
</View>

### Size

Buttons can be small, normal size, or large

<View style={{ flexDirection: 'row', alignItems: 'center'}}>
  <Button title="Small" size="small" style={{ marginHorizontal: 10 }} />
  <Button title="Default" size="default" style={{ marginHorizontal: 10 }} />
  <Button title="Large" size="large" style={{ marginHorizontal: 10 }} />
</View>

### State

While all the buttons above appear active, they can also be disabled and therefore uninteractive.

<View style={{ flexDirection: 'row', alignItems: 'center'}}>
  <Button title="Default" size="default" disabled={true} style={{ marginHorizontal: 10 }} />
</View>

### With Icons

<IconsStyle />

<View style={{ flexDirection: 'row', alignItems: 'center'}}>
  <Button title="Icon on the left" icon="icon-arrow-up" iconPosition="left" onPress={() => Alert.alert('Icon Button pressed')} style={{ marginHorizontal: 10 }} />
  <Button title="Icon on the right" icon="icon-arrow-right" iconPosition="right" onPress={() => Alert.alert('Icon Button pressed')} style={{ marginHorizontal: 10 }} />
</View>

## Props

:::note
Includes all [TouchableOpacityProps](https://reactnative.dev/docs/touchableopacity#props) props.
:::

<div className='table-responsive'>

| Name           | Type                                   | Default   | Description                                                                                                                                         |
| -------------- | -------------------------------------- | --------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `title`        | `string`                               |           | Text to appear on the button                                                                                                                        |
| `textStyle`    | TextStyle                              |           | Add additional styling for title component.                                                                                                         |
| `appearance`   | `filled` \| `outline` \| `transparent` | `filled`  | Fill type of the button.                                                                                                                            |
| `icon`         | IconNode                               |           | Displays an icon next to the button title. By default, it appears on the left of the title text. `iconPosition` can be used to specify its position |
| `iconPosition` | `left` \| `right`                      | `left`    | Displays Icon to the position mentioned. Needs to be used along with `icon` prop.                                                                   |
| `size`         | `default` \| `large` \| `small`        | `default` | Button size                                                                                                                                         |
| `fullWidth`    | boolean                                | `false`   | Makes the button full width.                                                                                                                        |
| `disabled`     | boolean                                | `false`   | Disables user interaction.                                                                                                                          |

</div>

```SnackPlayer name=Button%20Example
import React from 'react';
import {
  StyleSheet,
  View,
  SafeAreaView,
  Alert,
} from 'react-native';
import { Button } from 'etta-ui';

const Separator = () => <View style={styles.separator} />;

const App = () => (
  <SafeAreaView style={styles.container}>
    <View>
      <Button
        title="Normal"
        size="default"
        appearance="filled"
        onPress={() => Alert.alert('Simple Button pressed')}
      />
      <Separator />
       <Button
        title="With Icon"
        size="default"
        appearance="filled"
        icon="icon-arrow-up"
        iconPosition="left"
        onPress={() => Alert.alert('Icon Button pressed')}
      />
      <Separator />
      <Button
        title="Full width button"
        size="default"
        appearance="filled"
        icon="icon-arrow-up"
        iconPosition="left"
        onPress={() => Alert.alert('Icon Button pressed')}
        fullWidth={true}
      />
      <Separator />
      <Button
        title="Disabled button"
        size="default"
        appearance="filled"
        icon="icon-arrow-up"
        iconPosition="left"
        onPress={() => Alert.alert('Icon Button pressed')}
        disabled={true}
      />
    </View>
  </SafeAreaView>
);
const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    marginHorizontal: 16,
  },
  separator: {
    marginVertical: 8,
    borderBottomColor: '#737373',
    borderBottomWidth: StyleSheet.hairlineWidth,
  },
});
export default App;
```
