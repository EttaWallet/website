import { View } from 'react-native';
import { MnemonicQuiz } from 'etta-ui';

This element will help you build a manual seed phrase backup component with validation

## How to use it

```jsx
import { MnemonicQuiz } from 'etta-ui';

<MnemonicQuiz 
    seed="yam flower delight seed cat note gritty pirate vault bean time loud"
    numMnemonicChoices={8}
    checkingDuration={1.8*1000}
    resetTitle="Start over"
/>;
```

## Props

<div className='table-responsive'>

| Name           | Type                                   | Default   | Description                                                        |
| -------------- | -------------------------------------- | --------- | ------------------------------------------------------------------ |
| `seed`     | `string`        |           | The seed phrase as a string separated by spaces. E.g from BDK  |
| `numMnemonicChoices`     | `number`        |     6      | Number of mnemonic word choices to show in quiz  |
| `checkingDuration`     | `number`        |     1.8 * 1000      | Manual delay period for validating mnemonic  |
| `onPassed?`     | `function`: () => void          | Define what happens when user passes quiz |
| `onFail?`     | `function`: () => void          | Define what happens when user fails quiz |
| `resetTitle`     | `string` |          | Button title to reset quiz |

</div>
