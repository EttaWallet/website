Chips are small pressable elements used to display an attribute or initiate an action. They can display text and or icons.

## How to use it

```jsx
import { Chip } from 'etta-ui';

<Chip 
    selected={false} 
    icon="icon-bell" 
    iconPosition="right">
    Chip
</Chip>;
```

## Props

:::note
Includes all [TouchableOpacityProps](https://reactnative.dev/docs/touchableopacity#props) props.
:::


<div className='table-responsive'>

| Name           | Type                                   | Default   | Description                                                                                                                                         |
| -------------- | -------------------------------------- | --------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `children`        | `string`                                 |           | Text to display inside chip                                                                                                                        |
| `selected`     | `boolean`                            |   `false`        | Show chip as selected or not                                                                                                                        |
| `icon?`         | `bitcoin icon`                               |           | Displays an icon next to the chip text. By default, it appears on the left of the title. |
| `iconPosition?` | `left` \| `right`                      | `left`    | Specifies on which side the icon appears with respect to the chip text.                                                                   |

</div>
